# <p align="center">quick cobalt **>>**</p>
An extension that allows you to quickly open the active tab's URL in [cobalt.tools](https://cobalt.tools/) in order to download any present media.
To use, simply click the extension in your toolbar while in the tab you'd like to open in Cobalt.

## Usage
On **Firefox**, simply [add the addon](https://addons.mozilla.org/firefox/addon/quick-cobalt/).

On **Chrome and other Chromium based browsers**, you must download the code and [load the unpacked extension](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world#load-unpacked) (Developer mode must be on).
